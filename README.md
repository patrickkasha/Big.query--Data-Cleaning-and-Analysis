BigQuery-Data-Cleaning-and-Analysis

Overview

This repository includes SQL projects for cleaning, transforming and analyzing data with Google BigQuery. All the datasets used for the project were sourced from Kaggle and other open-source publicly available data sources. The purpose of this repository is to showcase both the practical aspect of SQL skills and real-world data analysis techniques via cloud-based tools and scalable query processing.


Topics Covered

The tasks in this repository focus on different methods of data cleaning and data analysis, such as:

* Handling missing and null values in SQL
* Removing duplicate data and identifying duplicate records
* Using data field types and converting them for analysis
* Removing white space from text fields and extracting substrings from text fields
* Formatting and normalizing text data into a common style
* Using joins to relate multiple datasets
* Creating Common Table Expressions (CTEs) for readable and reusable queries
* Creating temporary tables and views for analysis workflows
* Using window functions to rank, partition and perform calculations across rows
* Using aggregate functions such as `SUM`, `COUNT`, `AVG`, `MIN`, and `MAX` to summarize data

Usage

The SQL scripts included in this repository can be used by uploading the datasets to your Google BigQuery environment and running the included SQL queries. The scripts are designed to demonstrate both data cleaning workflows and analytical processes that can be applied to real-world datasets.



Purpose

The purpose of this repository is to provide a learning ground, build a portfolio, and practice advanced SQL skills for data analytics and cloud data processing.
