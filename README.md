BigQuery-Data-Cleaning-and-Analysis

This repository includes SQL projects for cleaning, transforming and analyzing data with Google BigQuery. All the datasets used for the project were sourced from Kaggle, other open source publicly available data sources. Purpose of this repository is to showcase both the practical aspect of SQL skills and real-world data analysis techniques via cloud-based tools and scalable query processing.

Topics Covered

The tasks in this repository focus on different methods of data cleaning and data analysis, such as:

This contains different approaches to dealing with missing and null values in SQL.
Removing duplicate data and identifying duplicate data.
* Using data field types and converting them for analysis.
* Removing white space from text fields and extracting substrings from text fields
Erodeing unwanted white space, formatting text data to a common style.Removing unused white space, normalizing text data.
Use joins to relate to multiple data sets
Creating Common Table Expressions (CTE) for easily read and reused queries
* Using integer data instead of decimal data
Using window functions to rank, partition and perform calculations over rows
Incorporate aggregate functions like SUM, COUNT, AVG, MIN and MAX to summarize data.

Tools Used

* Google BigQuery
* SQL
* Google Cloud Platform (GCP)

Usage

The SQL scripts included in this repository can be used by uploading the datasets to your Google BigQuery environment and running the included SQL queries. The scripts have been designed to follow data cleaning workflows as well as analysis scripts that can be used with real datasets.

The purpose of this repository is to provide a learning ground, build a portfolio, and to practice complex SQL skills for data analytics and cloud data processing.
